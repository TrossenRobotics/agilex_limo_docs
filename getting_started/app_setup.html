Mobile App SetupÔÉÅ

  1. Download and install the controller application to your mobile phone using the QR code below.

  2. Tap on the Bluetooth icon in the upper left to open the connection menu.

  3. Select the connection with the same name as the one on the front of your robot.

  4. Once connected, you will see the battery level of your robot as well as the connection symbol. You should also be able to control your robot. Find information on controlling your robot using the mobile app in the App Operation Guide.
